      Add new athlete

      To add a new athlete to the database, enter his Name, Position, Year, and Number to the right. The minimum fields needed to create a profile are: Name, Position, and Year. Click the "Save Athlete" button once you are done. You can use the "Clear Stats" button or refresh the page to start over.

      Edit saved athlete

      To edit a saved athlete, click "Saved Athletes" to pull up his profile. Note: if you change the athlete's Name or Year, you will create a new profile. Once you are satisfied with your changes, click "Save Athlete".

          About this project

          This interactive dashboard visualizes an athlete's physical attributes and off-season testing performance. Each athlete is benchmarked against six years of data from athletes in his position group. The dashboard provides multiple views of how an athlete compare to his peers in standardized testing.

          This project was developed for Penn Strength and Conditioning and Penn Football in fulfilment of the Master of Urban Spatial Analytics Practicum (MUSA-6110) in Spring 2025.

          Using this dashboard

          1 / Modify data

          Use the Add Data to add and update athlete profiles. Use the Saved Athletes button to load existing athlete profiles. You can change an athlete's demographics (name, number, position, year) and his testing stats. If you save a new name-position combination, a new athlete profile will be added to the database.

          2 / Visualize data

          On the Athlete Report tab, you can view an athlete's testing numbers and benchmark him against his position group. Barcharts visualize an athlete's percentile within his position, and are annotated with his testing stats. The radar chart summarizes an athlete's attributes relative to his position group. Hover over charts to see more details.

          3 / Compare athletes

          Use the Compare Athletes tab to compare two athletes side-by-side. Use the dropdown buttons to select who you'd like to compare. All available stats will be displayed in the table below. For tests that are populated for both athletes, a green check will indicate the superior athlete for each test.
